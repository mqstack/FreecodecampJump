User story

An informal, natural language description of one or more features of a software system

A story map in action

Here are examples:

  • Screening Quiz (Epic Story) As the HR manager, I want to create a screening quiz so that I can understand whether I want to send possible recruits to the functional manager.[9]
  • Quiz Recall As a manager, I want to browse my existing quizzes so I can recall what I have in place and figure out if I can just reuse or update an existing quiz for the position I need now.[9]
  • Limited Backup As a user, I can indicate folders not to backup so that my backup drive isn't filled up with things I don't need saved.[10]

There is no good evidence that using user stories increases software success or developer productivity. However, user stories facilitate sensemaking without undue problem structuring, which is linked to success
